### 实验内容
程序应有自己的文件根路径，即程序自己的一个包含所有操作文件的目录。
而程序应仅提供根路径的获取方法，屏蔽其创建方法，外部无需关心根路径的具体地址，仅需基于根路径完成文件操作。与Database类的设计相同。

说明
所有异常，全部在调用方法内捕获处理
应参考课程ppt讲义，而非github实验源码，实现

相对路径，不能使用“/”为前缀
